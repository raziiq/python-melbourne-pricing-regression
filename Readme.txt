This repo demonstrates the design of price prediction on Melbourne Housing Data. The repo goes into the detail of how to select the features, that includes, removing un-necessary features, performing cleaning, and using pipelines
Furthermore, once the score is received, Gradient Boosting is used to improve the score which is them blended with GridSearchCV to further improve the score.
The repo will walk you through each step with code examples to make it easier to understand.

The repo also contains the datasets used for practising.
